{% trans "Aparaty połówkowe" %}

{% blocktrans %} Aparaty połówkowe (ang. "half-frame camera) charakteryzują się tym, że wykorzystują tylko połowę domyślnego obszaru klatki dla danego typu negatywu. W przypadku najpopularniejszego formatu 135, z formatu klatki 24x36mm wykorzystywana jest pionowa połowa o rozmiarze 18x24mm. {% endblocktrans %}

{% blocktrans %} Pierwowzorem aparatów połówkowych był model 'Pen' japońskiej marki Olympus, a okres największej popularności to lata 60-70. Wraz z postępem miniaturyzacji aparatów kompaktowych o standardowym rozmiarze klatki 135 aparaty zaczęły odchodzić w zapomnienie. Największą zaletą tego typu aparatów jest możliwość wykonania dwukrotnie większej liczby ekspozycji na jednej rolce filmu, nawet ponad 72 zdjęcia dla filmu 36 klatkowego. {% endblocktrans %}

{% trans "Aparaty kompaktowe" %}

{% blocktrans %} Aparaty kompaktowe, potocznie kompakt (ang. "compact camera") – typ popularnego aparatu fotograficznego o niewielkich rozmiarach, zwartej budowie i uproszczonej obsłudze. Jest on najczęściej wyposażony w niewymienny obiektyw stałoogniskowy lub zmiennoogniskowy. {% endblocktrans %}

{% blocktrans %} Ustawienie ostrości dokonywane jest przez układ Auto Focus lub często manualnie na podstawie tabeli odległości lub piktogramów.
